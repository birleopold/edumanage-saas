About Performance Snapshots

Performance snapshots calculate and store comprehensive analytics for each student including:

  • GPA and overall percentage
  • Class rankings and percentiles
  • Subject-wise performance breakdown
  • Performance trends and predictions
  • At-risk student identification
  • Attendance and discipline metrics
