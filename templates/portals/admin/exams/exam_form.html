Exam setup checklist

Confirm these foundations before creating papers and schedules.

Choose the correct termEvery paper added later must use an offering from the same academic term.
Select the real delivery modePaper, online and hybrid modes affect the operational tools staff will prepare.
Use a realistic date windowThe start and end dates should cover the full examination operation, not only one paper.
Separate general and paper guidancePlace shared rules here; subject-specific instructions belong on the individual paper.
After saving the periodCreate exam papers, add or select questions, build room schedules, allocate seats, enter scores, review analytics and only then decide whether results are ready for publication.
